# Task 1: Scan Your Local Network for Open Ports

## Objective
The objective of this task is to discover active hosts and identify open ports on a local network using Nmap. This helps in understanding network exposure and commonly used network services.

## Tools Used
- Nmap
- Windows Command Prompt
- Windows Operating System

## Tasks Performed
- Identified the local IP address and network range.
- Performed a network discovery scan.
- Identified active hosts on the local network.
- Conducted a TCP port scan on the local machine.
- Analyzed the services associated with the open ports.
- Performed a basic security analysis of the identified services.

## Findings
- Five active hosts were identified on the local network.
- Seven open TCP ports were discovered on the local machine.
- Common services such as MSRPC, NetBIOS, Microsoft-DS, Oracle, and MySQL were identified.

## Conclusion
This task provided practical experience in network reconnaissance using Nmap and improved understanding of open ports, network services, and basic security considerations.
